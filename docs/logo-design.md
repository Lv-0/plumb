# Plumb Logo 设计文档

**日期**：2026-06-18（状态栏图标更新于 2026-07-17）
**状态**：App 图标已定稿；状态栏图标于 2026-07-17 更新

## 设计意图

**产品**：Plumb —— macOS 菜单栏窗口居中/平铺工具。名字取自"铅锤线"（plumb line），木匠用它找到真正的垂直与中心。

**核心隐喻**：一根线垂下一滴水滴，自然指向"正中"——正如窗口被温柔地安放到正确位置。

**形态灵感**：
- 《三体》中的"水滴"探测器：极致圆润、镜面般完美、无瑕的光滑曲面。
- 一颗完美水滴悬挂于深靛之中，由一线月光垂引，落定于正中。

## 视觉规范

### 形态（关键）
- **底部 2/3**：接近完美球面的饱满曲面，视觉上必须绝对圆润。
- **顶部 1/3**：与球面相切连续的曲线收束到一个优雅尖点（G1 连续，无折角）。
- 水滴尖端朝上、圆润底部朝下，整体居中、左右严格对称。

### 配色（墨夜月白 / Ink & Moonlight，经参考图调整为深靛版）
| 元素 | 颜色 | 说明 |
|------|------|------|
| 背景 | 深靛蓝 `#0A0E27` → `#050814` | 顶部略亮、底部更沉，深邃如夜空（比纯黑更冷、更蓝） |
| 水滴主体 | 半透明白 `rgba(247,252,255,0.95)` → 折射青蓝 `rgba(115,166,235,0.88)` | 上端近透明白，向下逐渐带折射感青蓝调，模拟水/玻璃介质的折射 |
| 光晕 | 青蓝 `#1E90FF` @ 35% → 透明 | 径向发散，让水滴"发光" |
| 高光 | 纯白 @ 92% → 透明 | 明亮月牙，落在水滴顶部偏左 1/3 处（光源左上方） |
| 边缘描边 | 冷青蓝 `rgba(140,191,250,0.70)` | 把半透明白从深靛底上"提"出来，强化折射玻璃边缘感 |
| 悬线 | 冷银白 `rgba(199,222,252,0.55)` | 极细、半透明，从画面顶端垂直垂到水滴尖端 |

### 材质光影
- 水滴读作抛光液态金属 / 折射玻璃质感：硬而光滑的曲面。
- 左上方一弯明亮纯白月牙高光（月光打在光滑曲面上的反射）。
- 水滴下方极淡冷青投影，使其略浮于背景之上。
- 边缘冷青蓝轮廓光，与深靛背景分离。

## 实现位置

图标构建位于 `scripts/generate_icons.sh`：
- App 图标以 `assets/AppIcon-base.png` 为设计稿，生成标准 `.icns` 尺寸集合。
- `drawStatusIcon(size:)` 以 Core Graphics 生成 128×128 菜单栏 template 图标：圆角屏幕框内，一个应用窗口严格落在垂直中轴上，直接表达“把窗口放到屏幕中央”。

状态栏图标不沿用 App 图标的字面水滴，也不采用需要解释的字母或抽象负形；它用屏幕、窗口和中轴三个具象元素直接说明产品功能。图形不包含渐变、阴影或微型细节；AppKit 通过 `NSImage.isTemplate` 自动适配浅色、深色和菜单高亮状态。
